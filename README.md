# Go 100 Tips

各Tipを個別に実行できるプロジェクトです。

## ディレクトリ構成

```
02-01/          # Chapter 2, Tip 1
├── main.go
02-02/          # Chapter 2, Tip 2
├── main.go
...
```

## 実行方法

各ディレクトリで以下のコマンドを実行してください：

```bash
# Tip 2-1を実行
cd 02-01
go run main.go

# Tip 2-2を実行
cd 02-02
go run main.go
```

## 新しいTipの追加

1. 新しいディレクトリを作成（例：`03-01`）
2. `main.go`ファイルを作成
3. 必要に応じてその他のGoファイルを追加

各プログラムは独立しているため、個別に実行・テストが可能です。
